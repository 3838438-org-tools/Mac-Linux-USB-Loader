Mac Linux USB Loader

This is a tool allowing you to put a Linux distro on a USB drive and make it bootable on Intel Macs using EFI. Right now, it's very basic and rudimentary but is being very actively developed. Some advanced features are being developed for Mac Linux USB Loader that are not available in other live USB creators for Mac, such as the ability to make the Live USB drives created by this tool automatically boot on your Mac, a Compatibility Tester application, and more!

The application is available in English and is partially translated into Spanish. Most user interface elements are translated, but dialog boxes and recent additions (particularly in development branches) remain in English.

I recommend that you download the latest version (link below), but if you want to build from source code, use the following steps (*Note:* more thorough steps on building from source are here).

mkdir whatever
cd whatever
git clone git://github.com/SevenBits/Mac-Linux-USB-Loader.git
xcodebuild

Please note: if you get an error about the certificate "Mac Developer" not being present, this means that you are not enrolled in the Mac Developer Program. Code signing is enabled as a benefit to Mac developers. If you are not enrolled as a paid member of the Mac Developer Program, use xcodebuild CODE_SIGN_IDENTITY="" CODE_SIGNING_REQUIRED=NO instead of xcodebuild as shown above.

You can also go into the Xcode project and build from there, but that makes a debug build that is not useful unless you intend you help develop the application.
