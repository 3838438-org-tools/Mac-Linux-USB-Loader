Mac Linux USB Loader

This is a tool allowing you to put a Linux distro on a USB drive and make it bootable on Intel Macs using EFI. Right now, it's very basic and rudimentary but is being very actively developed.

To build it, perform the following steps from the command line:

mkdir whatever
cd whatever
git clone git@github.com:SevenBits/Mac-Linux-USB-Loader.git
xcodebuild

You can also go into the Xcode project and build from there, but that makes a debug build that is not useful unless you intend you help develop the application.

You can also download pre-build binaries here, though there is a close to 100% chance that it is out of date. Click here for the most recent build (I'll try to keep that up to date ;) ). Building the program yourself is the best way to see the new features.
